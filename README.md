# Wikipedia Search

This is a Wikipedia Search react web app.

Clone the repo to get started:

```properties
git clone https://github.com/donaldp/wikipedia-search
```

> `cd` into folder

Install npm dependencies:
```
yarn install
```

Finally, run the web app: 
```
yarn start
```
